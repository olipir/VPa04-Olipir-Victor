## День 1. Что такое SQL и реляционные базы данных

1. **Что такое реляционная база данных?**
   Это база данных, которая хранит информацию в виде таблиц (отношений), состоящих из строк и столбцов. Таблицы могут быть связаны между собой через ключи.

2. **Из каких элементов состоит таблица?**
   Из строк (записей, records) и столбцов (полей, columns). Каждый столбец имеет имя и тип данных.

3. **Как расшифровывается SQL и для чего он нужен?**
   SQL — Structured Query Language (язык структурированных запросов). Он нужен для взаимодействия с реляционными базами данных: чтения, вставки, обновления, удаления данных и управления структурой БД.

4. **Назови хотя бы две популярные реляционные СУБД.**
   PostgreSQL, MySQL, Microsoft SQL Server, Oracle Database, SQLite. (Любые две.)
